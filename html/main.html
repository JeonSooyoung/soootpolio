  1. 평소에 별로 SEO같은 걸 신경쓰지 못한게 아쉽다. 사실 검색 기반의 사이트를 만드는 게 아니라 기존의 사이트 유지보수정도 한다고 생각하니 별로 신경 쓰지 않았다. 이번 기회에 좀 개념을 다져서 검색 친화적인 사이트에 대한 개념 기반을 다지고 싶다.
    검색 최적화 작업
    검색엔진이 내 컨텐츠 정보를 잘 이해할 수 있도록 정리 작업
    이 작업을 통해서 사이트 내 콘텐츠가 네이버의 검색 결과에 누락되지 않도록 조치할 수 있다.
  2. 내가 약관 같은 거 작업 많이 하는 데 거기서 사용하는 클래스 나 이런 작업물이 공통화가 안되서 할때마다 매우 헷갈린다 이걸 공통화 해서 뭔가 작업하면 편할 거 같다.
  3. 그리고 CSS 쓸때마다 느끼는 건데 스타일 시트 열어서 쓰는 거 너무 불편하다 그냥 태그에 인라인 방식으로 박아 넣고 싶다는 생각 많이 한다 그렇게 넣어서 나중에 한꺼번에 변환 시키고 싶다. 굳이 이 위치 찾아서 스타일 넣는게 너무 귀찮다.
